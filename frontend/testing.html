Steps to Pay
  • Press *150*00# then ok.
  • Go to M-Pesa menu and select option number 4 (Pay by M-Pesa).
  • Go to option number 4 (enter business number): 711788.
  • Key in Savanna Fibre account number and send.
  • Add amount.
  • Enter Personal PIN.
  • Confirm payment.
  • Press *150*60# then Ok using Airtel line.
  • Go to option number 5 (Make payment).
  • Go to option number 4 (enter business number): 711788.
  • Enter reference number (your Savanna Fibre account number).
  • Enter amount.
  • Enter Personal PIN.
  • Confirm payment.
  • Press *150*01# then ok using the Tigo line.
  • Go to Tigo Pesa menu and select option number 4 (Pay bill).
  • Go to option number 3 (enter business number): 711788.
  • Key in Savanna Fibre account number and send.
  • Add amount.
  • Enter Personal PIN.
  • Confirm payment.

Please select a payment method to see the steps.
